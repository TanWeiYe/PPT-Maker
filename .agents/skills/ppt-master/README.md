## Usage

1. Read `SKILL.md` first to understand the full process.
2. Prepare environment variables from `.env.example` as needed.
3. Follow `SKILL.md` and run the needed scripts in `scripts/`.
4. If you need full `templates/` or `references/` assets, fetch them from upstream repository paths documented in each directory's `README.md`.
